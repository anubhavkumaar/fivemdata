# FiveM Server Dashboard

## Overview
The FiveM Server Dashboard is a web application that provides real-time information about a FiveM server. It allows users to view server details, search for players, and set alerts for specific players.

## Features
- Displays server information such as IP address and player count.
- Search functionality to filter players by name.
- Alert management to notify users when specific players are online.

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/fivem-server-dashboard.git
   ```
2. Navigate to the project directory:
   ```
   cd fivem-server-dashboard
   ```
3. Open `index.html` in your web browser to view the dashboard.

## Usage
- Use the search bar to find players by name.
- Add player names to the alert list to receive notifications when they are online.
- Sort players by name or ID using the dropdown menu.

## Deployment
This project can be deployed on Vercel. Ensure that the `vercel.json` file is configured correctly for your deployment settings.
